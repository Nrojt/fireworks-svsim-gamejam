# SVSIM 2026 game jam submission.
My game jam submission for [the 2026 SVSIM game jam](https://itch.io/jam/sim-game-jam-26).
This is a physics-based puzzle game where you have to destroy obstacles by placing fireworks in the right spots.

The orange ball is the ignition. Place fireworks to cause a chain reaction to destroy as many obstacles as possible.

The game is janky and unpolished as hell, since I sadly did not have/make enough time.

## Controls:
- LMB: select and place fireworks
- RMB: remove a placed firework
- Scroll Wheel: rotate the firework
- SPACE: ignite

You can hover over the shop items to get a short description of the firework.
